# Layer 1: 数值计算（通用）

> 适用范围：所有角色、所有流派

## 伤害计算公式

### 基础公式

```
最终伤害 = 基础伤害 + 力量
```

- 力量可为正可为负
- 最终伤害最小为 0

### 多段攻击

```
总伤害 = (基础伤害 + 力量) × 攻击次数
```

示例：
- 连击（基础伤害 5，攻击 2 次）
- 力量 = 3 时：总伤害 = (5+3) × 2 = 16

### 伤害乘数

```
最终伤害 = (基础伤害 + 力量) × 乘数效果

乘数效果：
- 脆弱（敌人）: ×1.5
- 虚弱（自己）: ×0.75
- 双持: ×2
```

### 穿甲计算

```
实际伤害 = 计算伤害 - 敌人格挡

注意：敌人格挡只在当回合有效
```

## 格挡计算

### 基础公式

```
格挡值 = 基础格挡 + 灵巧
```

- 灵巧来自遗物或卡牌效果
- 格挡在回合结束时清零

### 格挡乘数

```
最终格挡 = (基础格挡 + 灵巧) × 乘数

乘数效果：
- 脆弱（自己）: ×1.5 受伤，格挡价值提升
- 双持: ×2 格挡值
```

## 能量计算

### 能量来源

- 基础能量：3（每回合）
- 遗物加成：工具箱（+1）等
- 卡牌加成：祭品、燃烧契约等
- 药水加成：能量药水

### 能量效率

```
能量效率 = 卡牌价值 / 消耗能量

高效率操作：
- 0费牌：效率无限
- 过牌：抽到的牌的总价值 / 消耗能量
```

## 药水计算

| 药水 | 效果 | 价值评估 |
|------|------|----------|
| 力量药水 | +2力量 | 相当于每张攻击牌+2伤害 |
| 格挡药水 | +12格挡 | 保命用 |
| 能量药水 | +2能量 | 高费卡组价值高 |
| 液态记忆 | 复制一张手牌 | 关键牌×2 |
| 火焰药水 | 20伤害 | 紧急输出 |

## 边界条件

### 数值边界

| 属性 | 最小值 | 最大值 |
|------|--------|--------|
| 伤害 | 0 | 无上限 |
| 格挡 | 0 | 无上限 |
| 手牌 | 0 | 10 |
| 能量 | 0 | 无上限 |
| 抽牌堆 | 0 | 无上限 |
| 弃牌堆 | 0 | 无上限 |

### 关键阈值

- HP ≤ 0：死亡
- 能量 < 费用：无法打出
- 手牌 = 10：无法抽牌
- 抽牌堆 = 0：弃牌堆洗入
