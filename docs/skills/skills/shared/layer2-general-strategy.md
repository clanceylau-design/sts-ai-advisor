# Layer 2: 局部策略（通用）

> 适用范围：所有角色、所有流派
> 范式：启发式规则

## 回合内决策流程

### 决策优先级

```
1. 致死判断 → 必须防御
2. 杀敌判断 → 计算最优输出
3. 资源优化 → 过牌/上Buff/留存
```

### Step 1: 致死判断

```
IF 敌人下回合伤害 >= 当前HP - 当前格挡:
    优先级 = "防御"
    计算需要的最小格挡
    选择能提供该格挡的最优牌组合
```

**判断要素：**
- 敌人意图（攻击/强化/debuff）
- 敌人伤害值（基础+状态+意图）
- 自己HP和格挡
- 手牌中的防御牌

### Step 2: 杀敌判断

```
IF 本回合能打死敌人:
    计算所需伤害
    选择能量效率最高的输出组合
```

**注意：**
- 敌人可能有护甲
- 计算是否需要穿透
- 评估杀敌后的收益（战斗结束）

### Step 3: 资源优化

```
IF 无致死/杀敌威胁:
    评估手牌质量
    决定：过牌/上Buff/留存关键牌
```

## 出牌优先级规则

### 防御优先规则

| 条件 | 策略 | 优先级 |
|------|------|--------|
| 敌人下回合伤害致死 | 必须防御，计算最小格挡 | 最高 |
| 敌人下回合伤害>30%HP | 优先防御 | 高 |
| 有脆弱debuff | 格挡需求×1.5 | 高 |
| 敌人有多段攻击 | 每段都要覆盖格挡 | 高 |
| 敌人意图不明 | 保守估计，保留防御 | 中 |

### 输出优先规则

| 条件 | 策略 |
|------|------|
| 能量≥3 | 优先高费牌（通常效率更高） |
| 能量=1 | 优先0费牌 |
| 有力量加成 | 多段攻击牌优先 |
| 敌人有护甲 | 考虑穿透/重攻击 |
| 有双持 | 评估最优目标 |

### 过牌时机

| 条件 | 是否过牌 |
|------|----------|
| 手牌质量差 + 有过牌 | 是 |
| 手牌已有核心牌 | 否（可能把核心牌洗回去） |
| 能量紧张 | 否 |
| 有"牌位"遗物（陀螺） | 优先过牌 |
| 抽牌堆快空 | 是（即将洗牌） |

## 状态管理

### Debuff 处理优先级

| Debuff | 影响 | 处理优先级 |
|--------|------|-----------|
| 虚弱 | 输出×0.75 | 高（输出流派） |
| 脆弱 | 受伤×1.5 | 高（生存关键） |
| 灼烧 | 每回合掉血 | 中 |
| 易伤 | 受伤×1.5 | 中 |
| 弱点 | 手牌-1 | 低 |

**处理方式：**
- 有净化牌时评估价值
- 考虑敌人debuff持续时间
- 评估debuff的实际影响

### Buff 施加时机

| Buff | 施加时机 |
|------|----------|
| 力量 | 输出回合前 |
| 格挡 | 防御回合前 |
| 能量 | 手牌多时 |
| 抽牌 | 需要找关键牌时 |

## 能量管理

### 能量分配原则

1. **必须的牌优先**：防御/生存牌
2. **高效率牌次之**：0费牌、高价值牌
3. **可选牌最后**：过牌、非关键Buff

### 能量留存

| 情况 | 是否留存能量 |
|------|-------------|
| 手牌已出完 | 否 |
| 有0费牌可出 | 否 |
| 下回合有高费关键牌 | 可能留存 |
| 有能量遗物（日晷） | 是 |

---

## 手牌管理

### 手牌质量评估

```
手牌价值 = Σ(每张牌的即时价值)

即时价值取决于：
- 当前场景是否需要
- 能否打出（能量是否够）
- 是否有协同效果
```

### 留牌策略

| 情况 | 留什么牌 |
|------|----------|
| 回合结束 | 关键牌、0费牌、下回合需要的防御牌 |
| Boss战 | 爆发牌留到关键时刻 |
| 多波战斗 | 留AOE牌给下一波 |
