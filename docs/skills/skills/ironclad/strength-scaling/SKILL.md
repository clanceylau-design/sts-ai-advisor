# 力量叠加流 (Strength Scaling)

## 流派概述

**核心机制**: 力量 × 多段攻击 = 指数级伤害增长

**价值函数**: 攻击牌价值 = 基础伤害 + 力量 × 攻击次数

**风险等级**: 高（前期脆弱，启动慢）

---

## Layer 1: 数值计算（流派特化）

### 力量伤害公式

```
单次攻击伤害 = 基础伤害 + 力量值

多段攻击总伤害 = (基础伤害 + 力量值) × 攻击次数
```

**关键洞察**: 力量对多段攻击的收益是乘法的！

示例：
| 牌 | 基础伤害 | 力量=0 | 力量=5 | 力量=10 |
|---|---------|--------|--------|---------|
| 重刃 | 5+5×力量 | 5 | 30 | 55 |
| 连击 | 5×2次 | 10 | 20 | 30 |
| 剑柄打击 | 3+力量 | 3 | 8 | 13 |

### 力量来源计算

| 来源 | 力量值 | 备注 |
|------|--------|------|
| 工具箱（遗物） | +1 | 永久 |
| 突破极限 | +2 | 需要手牌≥1 |
| 恶魔形态 | +3/回合 | 需要存活 |
| 力量药水 | +2 | 临时 |
| 燃烧 | +1 | 需要手牌≥1 |

### 能量效率分析

力量流卡组的能量效率随力量增长：

```
能量效率 = (基础伤害 + 力量×攻击次数) / 费用
```

高力量时，低费攻击牌效率极高。

## Layer 2: 局部策略（流派特化）

### 力量施放时机

```
IF 手牌有多段攻击牌 AND 有力量来源:
    优先施放力量 → 然后输出
ELSE:
    按普通策略出牌
```

### 出牌顺序优化

**理想顺序**:
1. 力量来源牌（突破极限、恶魔形态）
2. 过牌找核心输出牌
3. 多段攻击输出

**注意事项**:
- 突破极限需要手牌≥1，留一张牌
- 恶魔形态需要存活，确保防御

### 防御策略

力量流防御薄弱，需要特别注意：

| 情况 | 策略 |
|------|------|
| HP>50% | 可以承受一些伤害，优先输出 |
| HP≤30% | 必须防御，存活优先 |
| 有多层小怪 | AOE优先或逐个击杀 |

### 过牌策略

| 过牌牌 | 使用时机 |
|--------|----------|
| 战吼 | 找力量牌或输出牌 |
| 燃烧契约 | 烧掉废牌，换核心牌 |
| 愤怒 | 需要爆发时 |

### 典型范例

#### 范例 1：成型力量流

```
卡组（18张）：
  核心：突破极限×2, 双持×1, 重刃×2
  攻击：剑柄打击×2, 连击×1
  防御：幽灵铠甲×2, 火焰护盾×1
  过牌：战吼×2, 燃烧契约×1
  其他：恶魔之火×1, 祭品×1

遗物：工具箱, 陀螺

分析：
- 成型信号：2张突破极限 + 重刃，力量流确认
- 牌位：18张，偏少，过牌够用
- 风险：防御薄弱，需要快速结束战斗
- 预期输出：力量=6时，重刃伤害=35
```

#### 范例 2：未成型但有潜力

```
卡组（24张）：
  核心：突破极限×1
  攻击：重刃×1, 连击×1, 打击×4
  防御：防御×4, 幽灵铠甲×1
  其他：...

分析：
- 信号：有突破极限+重刃，但只有1张力量来源
- 建议：继续寻找力量牌，暂时按普通流派打
- 选牌：力量牌优先级↑↑
- 风险：转型期最危险，防御要跟上
```

#### 范例 3：反例 - 混流派问题

```
卡组（22张）：
  力量组件：突破极限×2, 重刃×1
  防御组件：岿然不动×2, 幽灵铠甲×2
  其他：...

问题分析：
- 同时走力量流和防御流，资源分散
- 岿然不动不吃力量，在力量流中价值下降
- 建议：专注一边，放弃岿然不动或放弃力量

正确做法：
- 如果决定力量流：删掉岿然不动，加力量牌
- 如果决定防御流：删掉突破极限/重刃，加格挡牌
```

### 选牌决策

#### 看到这些牌时

| 牌 | 是否选择 | 条件 |
|---|---------|------|
| 突破极限 | **必拿** | 已经决定走力量流 |
| 双持 | **必拿** | 有力量牌时 |
| 重刃 | **优先** | 有力量来源时 |
| 连击 | **优先** | 有力量来源时 |
| 剑柄打击 | **优先** | 优质攻击+过牌 |
| 恶魔形态 | **谨慎** | 高费，启动慢 |
| 全能 | **不拿** | 已经走力量流 |
| 岿然不动 | **谨慎** | 防御会分散资源 |

#### 删除牌优先级

1. 诅咒牌
2. 打击（基础牌，效率低）
3. 防御（如果走纯力量流）
4. 与流派不兼容的牌
5. 低效率攻击牌

### Boss 准备

#### Time Eater（时间吞噬者）

**克制关系**: 不利

**原因**:
- 12张牌触发回血+加力量
- 力量流通常需要多回合输出
- 回合数越多越危险

**应对**:
- 控制出牌数≤11/回合
- 爆发回合提前规划
- 考虑走更快的流派

#### The Heart（心脏）

**克制关系**: 不利

**原因**:
- 第2回合开始每回合打108血
- 需要持续防御+输出
- 力量流启动慢

**应对**:
- 确保有足够防御牌
- 力量药水预留
- 考虑幽灵铠甲保命

### 商店策略

**优先级**:
1. 删除打击/防御（减牌位）
2. 买力量相关遗物（工具箱、陀螺）
3. 买力量牌（突破极限、双持）
4. 买力量药水

## 关键牌详解

### 突破极限 (Limit Break)

```
费用: 1
效果: 将你的力量翻倍
升级: 返回手牌
```

**使用要点**:
- 升级后可循环使用
- 需要先有力量基础
- 手牌≥1时使用

**价值**: 本流派核心，价值★★★★★

### 重刃 (Heavy Blade)

```
费用: 2
效果: 造成5点伤害，每点力量额外造成5点伤害
升级: 力量加成变为7
```

**使用要点**:
- 力量≥3时开始超值
- 升级后价值大幅提升
- 配合突破极限爆炸输出

**价值**: 力量流核心输出牌，价值★★★★★

### 双持 (Dual Wield)

```
费用: 1
效果: 选择一张攻击牌，在本场战斗中，该牌的数量翻倍
升级: 改为复制两张
```

**使用要点**:
- 复制突破极限：力量翻倍翻倍
- 复制重刃：输出翻倍
- 复制防御牌：保命

**价值**: 核心辅助牌，价值★★★★★
