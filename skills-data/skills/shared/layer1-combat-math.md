# Layer 1: 数值计算（通用）

> 适用范围：所有角色、所有流派
> 来源：攻略文档整理

## 伤害计算公式

### 基础公式

```
最终伤害 = 基础伤害 + 力量
```

- 力量可为正可为负
- 最终伤害最小为 0

### 多段攻击

```
总伤害 = (基础伤害 + 力量) × 攻击次数
```

**关键洞察**: 力量对多段攻击的收益是乘法的！

示例：
| 牌 | 基础伤害 | 力量=0 | 力量=5 | 力量=10 |
|---|---------|--------|--------|---------|
| 重刃 | 14+3×力量 | 14 | 29 | 44 |
| 连击 | 5×2次 | 10 | 20 | 30 |
| 铁斩波 | 5 | 5 | 10 | 15 |

### 伤害乘数

```
最终伤害 = (基础伤害 + 力量) × 乘数效果

乘数效果：
- 脆弱（敌人）: ×1.5
- 虚弱（自己）: ×0.75
- 双持: ×2
```

### 穿甲计算

```
实际伤害 = 计算伤害 - 敌人格挡

注意：敌人格挡只在当回合有效
```

## 格挡计算

### 基础公式

```
格挡值 = 基础格挡 + 灵巧/敏捷
```

- 灵巧/敏捷来自遗物或卡牌效果
- 格挡在回合结束时清零（除非有壁垒）

### 格挡乘数

```
最终格挡 = (基础格挡 + 灵巧) × 乘数

乘数效果：
- 双持: ×2 格挡值
- 巩固: ×2 格挡值
```

## 能量计算

### 能量来源

- 基础能量：3（每回合）
- 遗物加成：墨水（+1）、怀表（每3回合+2）
- 卡牌加成：祭品、燃烧契约等

### 能量效率

```
能量效率 = 卡牌价值 / 消耗能量

高效率操作：
- 0费牌：效率无限
- 过牌：抽到的牌的总价值 / 消耗能量
```

## 抽牌机制

### 基础规则

- 每回合开始抽5张牌
- 手牌上限为10张
- 超过上限的抽牌进入弃牌堆

### 洗牌循环

```
抽牌堆 → 手牌 → 弃牌堆 → 洗牌 → 抽牌堆
```

- 当抽牌堆空时，弃牌堆洗入抽牌堆
- 消耗牌（如燃烧）不进入弃牌堆，进入消耗堆
- 日晷在洗牌时触发效果（获得2点能量）

## 卡组管理

### 理想卡组数量

| 类型 | 数量 | 说明 |
|------|------|------|
| 普通卡组 | 20-30张 | 平衡稳定性与灵活性 |
| 无限卡组 | 5-10张 | 精简，需要特定遗物支持 |

### 卡组构成比例

| 类型 | 占比 | 说明 |
|------|------|------|
| 攻击牌 | 40-50% | 输出核心 |
| 防御牌 | 20-30% | 生存保障 |
| 技能牌 | 20-30% | 功能支持 |
| 能力牌 | 10-20% | 持续效果 |

### 能量曲线

| 费用 | 定位 | 说明 |
|------|------|------|
| 0费 | 填充 | 填充能量空缺，无限流核心 |
| 1费 | 主力 | 占比最高，能量效率好 |
| 2费 | 核心 | 强力输出或防御 |
| 3费+ | 强力 | 效果强大，但使用受限 |

## 过渡期伤害计算

### 一阶段需求

- BOSS血量约75左右
- 需要在3回合爆发打出约60点伤害
- 理论上需要增加2张1费15伤卡，或1张爆发30伤卡

### 二阶段需求

- 怪物血量和伤害提高
- 需求大于一阶段一倍
- 伤害需求至少还需再增加约60伤

### 伤害牌价值评估

```
负面卡收益 = 能量和伤害的收益比

示例：
1费打15伤（狂野打击）需要承受副作用
需要考虑卡组循环的负面收益
```

## 边界条件

### 数值边界

| 属性 | 最小值 | 最大值 |
|------|--------|--------|
| 伤害 | 0 | 无上限 |
| 格挡 | 0 | 无上限 |
| 手牌 | 0 | 10 |
| 能量 | 0 | 无上限 |

### 关键阈值

- HP ≤ 0：死亡
- 能量 < 费用：无法打出
- 手牌 = 10：无法抽牌
- 抽牌堆 = 0：弃牌堆洗入
