# 力量叠加流 (Strength Scaling)

## 流派概述

**核心机制**: 力量 × 多段攻击 = 指数级伤害增长

**价值函数**: 攻击牌价值 = 基础伤害 + 力量 × 攻击次数

**风险等级**: 高（前期脆弱，启动慢）

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 力量伤害公式

```
单次攻击伤害 = 基础伤害 + 力量值

多段攻击总伤害 = (基础伤害 + 力量值) × 攻击次数
```

**关键洞察**: 力量对多段攻击的收益是乘法的！

示例：
| 牌 | 基础 | 力量=0 | 力量=5 | 力量=10 |
|---|------|--------|--------|---------|
| 重刃 | 14+3×力量 | 14 | 29 | 44 |
| 连击 | 5×2次 | 10 | 20 | 30 |
| 剑柄打击 | 10+力量 | 10 | 15 | 20 |

### 力量来源

| 来源 | 力量值 | 备注 |
|------|--------|------|
| 活动肌肉 | +2（临时） | 回合结束失去 |
| 恶魔形态 | +2/回合 | 需存活，启动慢 |
| 突破极限 | 力量翻倍 | 核心卡牌 |
| 观察弱点 | +2 | 敌人攻击意图时 |
| 工具箱（遗物） | +1 | 永久 |
| 红骷髅（遗物） | +3 | HP≤50%时 |

### 能量效率分析

力量流卡组的能量效率随力量增长：

```
能量效率 = (基础伤害 + 力量) / 费用
```

高力量时，低费攻击牌效率极高。

## Layer 2: 局部策略（流派特化）

### 力量施放时机

```
IF 手牌有多段攻击牌 AND 有力量来源:
    优先施放力量 → 然后输出
ELSE:
    按普通策略出牌
```

### 出牌顺序优化

**理想顺序**:
1. 活动肌肉（获得临时力量）
2. 突破极限（力量翻倍）
3. 双持复制重刃/连击
4. 输出

**注意事项**:
- 突破极限需要当前有力量才能翻倍
- 活动肌肉的力量是临时的，回合结束失去
- 双持升级后变为0费，效率更高

### 防御策略

力量流防御薄弱，需要特别注意：

| 情况 | 策略 |
|------|------|
| HP>50% | 可以承受一些伤害，优先输出 |
| HP≤30% | 必须防御，存活优先 |
| 有多层小怪 | AOE优先或逐个击杀 |

### Boss 准备

#### 时间吞噬者

**克制关系**: 不利

**原因**:
- 12张牌触发回血+加力量
- 力量流通常需要多回合输出
- 回合数越多越危险

**应对**:
- 控制出牌数≤11/回合
- 爆发回合提前规划

#### 心脏

**克制关系**: 不利

**原因**:
- 第2回合开始每回合打108血
- 需要持续防御+输出
- 力量流启动慢

**应对**:
- 确保有足够防御牌
- 力量药水预留

### 商店策略

**优先级**:
1. 删除打击/防御（减牌位）
2. 买力量相关遗物（工具箱、陀螺、红骷髅）
3. 买力量牌（突破极限、双持）
4. 买力量药水

## 关键牌详解

### 突破极限 (Limit Break)

```
费用: 1
效果: 将你的力量翻倍
升级: 费用变为0费
```

**使用要点**:
- 升级后变为0费，可循环使用
- 需要先有力量基础才能翻倍
- 配合活动肌肉：临时力量也能翻倍

**价值**: 本流派核心，价值★★★★★

### 重刃 (Heavy Blade)

```
费用: 2
效果: 造成14点伤害，力量对其造成3倍伤害
升级: 基础伤害提升至18
```

**使用要点**:
- 力量≥3时开始超值
- 升级后基础伤害更高
- 配合突破极限爆炸输出

**价值**: 力量流核心输出牌，价值★★★★★

### 双持 (Dual Wield)

```
费用: 1
效果: 将手牌中的一张攻击牌复制一张放入手牌
升级: 费用变为0费
```

**使用要点**:
- 复制突破极限：力量翻倍再翻倍
- 复制重刃：输出翻倍
- 复制防御牌：保命

**价值**: 核心辅助牌，价值★★★★★

### 活动肌肉 (Flex)

```
费用: 1
效果: 获得2点力量，在本回合结束时失去2点力量
升级: 获得3点力量
```

**使用要点**:
- 配合突破极限：临时力量也能翻倍
- 先打活动肌肉，再突破极限
- 力量变成永久

**价值**: 配合牌，价值★★★★☆
