# 灼热流 (Searing Deck)

## 流派概述

**核心机制**: 灼热攻击可无限升级，每升级+3伤害

**价值函数**: 灼热伤害 = 7 + 升级次数×3

**风险等级**: 高

**来源**: 攻略文档整理

**评价**: 费力不讨好，吃篝火次数，遗物依赖

---

## Layer 1: 数值计算（流派特化）

### 灼热攻击机制

```
灼热攻击（1费）:
- 造成7点伤害
- 可被无限升级
- 每次升级+3伤害

升级后:
- 10伤害（+1次）
- 13伤害（+2次）
- 16伤害（+3次）
```

### 升级来源

| 方式 | 效果 |
|------|------|
| 篝火 | 升级1张牌 |
| 武装 | 升级所有攻击牌 |
| 铁锤遗物 | 每次战斗升级1张 |

### 伤害成长

| 升级次数 | 伤害 | 费效比 |
|----------|------|--------|
| 0 | 7 | 7/费 |
| 2 | 13 | 13/费 |
| 5 | 22 | 22/费 |
| 10 | 37 | 37/费 |

## Layer 2: 局部策略（流派特化）

### 启动条件

```
必要条件:
1. 灼热攻击
2. 篝火次数（至少4次）
3. 武装（加速升级）

理想条件:
- 铁锤遗物
- 多个篝火
```

### 升级策略

```
优先级:
1. 用篝火升级灼热攻击
2. 打出武装升级攻击牌
3. 狂怒配合打出更多攻击牌

注意:
- 灼热攻击是固定2费消耗
- 需要大量篝火次数
- 前期容易暴毙
```

### 狂怒配合

```
狂怒（0费）:
- 造成3点伤害
- 本回合每打出一张攻击牌，伤害+2

配合:
- 灼热攻击+狂怒轮转
- 每回合打出灼热攻击升级
```
