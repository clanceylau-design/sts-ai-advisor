# 防战流 (Block Deck)

## 流派概述

**核心机制**: 高格挡 + 壁垒/岿然不动 = 无敌防御，以守为攻

**价值函数**: 生存优先，通过全身撞击输出

**风险等级**: 低（稳定性高）

**来源**: 攻略文档整理

**评价**: 战士第一流派，新手必学

---

## Layer 1: 数值计算（流派特化）

### 格挡计算公式

```
基础格挡 = 牌面格挡 + 灵巧

最终格挡 = 基础格挡 × 乘数效果
```

### 格挡来源

| 来源 | 格挡值 | 备注 |
|------|--------|------|
| 防御 | 5 | 基础牌 |
| 耸肩无视 | 8 | +抽1牌 |
| 铁斩波 | 4 | +伤害5 |
| 幽灵铠甲 | 每回合1-2 | 能力牌 |
| 无惧疼痛 | 受伤时1-2 | 能力牌 |
| 巩固 | 格挡×2 | 核心乘数 |
| 壁垒 | 格挡不消失 | 核心能力 |

### 格挡翻倍效应

```
回合1: 10格挡
巩固后: 20格挡
壁垒后: 保留到下回合
回合2: 20+10=30格挡
巩固后: 60格挡
```

### 能量效率分析

防战流能量效率稳定：
- 1费获得8格挡（耸肩无视）= 效率8
- 巩固2费格挡翻倍 = 效率取决于当前格挡

## Layer 2: 局部策略（流派特化）

### 防御优先级

```
1. 确保存活（格挡≥敌人伤害）
2. 积累格挡（使用低费格挡牌）
3. 翻倍格挡（巩固）
4. 输出（全身撞击）
```

### 出牌顺序

**理想顺序**:
1. 使用格挡牌积累基础格挡
2. 使用巩固翻倍格挡
3. 使用全身撞击输出
4. 如有壁垒，格挡保留到下回合

### 进攻策略

| 情况 | 策略 |
|------|------|
| 格挡≥敌人伤害 | 可以输出 |
| 格挡<敌人伤害 | 继续积累格挡 |
| 有全身撞击 | 格挡越高伤害越高 |

### 过牌策略

| 过牌牌 | 使用时机 |
|--------|----------|
| 耸肩无视 | 需要8格挡+找牌 |
| 战斗专注 | 需要大量过牌 |

## 关键牌详解

### 无惧疼痛 (Feel No Pain)

```
费用: 1
效果: 每当你受到攻击伤害，获得1点格挡
升级: 获得2点格挡
```

**使用要点**:
- 被动叠甲，敌人攻击越多格挡越多
- 配合高格挡形成良性循环
- 启动只需要1费

**价值**: 防战核心，价值★★★★★

### 壁垒 (Barricade)

```
费用: 3
效果: 格挡不再在回合开始时消失
升级: 费用变为2费
```

**使用要点**:
- 格挡可以跨回合积累
- 配合巩固实现指数增长
- 升级后更容易启动

**价值**: 防战核心，价值★★★★★

### 巩固 (Body Slam)

```
费用: 2
效果: 将你当前的格挡翻倍
升级: 费用变为1费
```

**使用要点**:
- 格挡越高收益越大
- 可以多次使用
- 升级后1费翻倍

**价值**: 格挡乘数核心，价值★★★★★

### 全身撞击 (Body Slam)

```
费用: 1
效果: 造成等同于当前格挡值的伤害
升级: 基础伤害提升
```

**使用要点**:
- 格挡越高伤害越高
- 低费高效
- 可以一回合打出多次

**价值**: 防战核心输出，价值★★★★★
