# 腐化流 (Corruption Deck)

## 流派概述

**核心机制**: 腐化 + 黑暗之拥 = 技能牌0费 + 消耗抽牌 = 伪无限循环

**价值函数**: 技能牌能量效率无限大

**风险等级**: 中等

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 腐化效果

```
腐化（2费能力牌）:
- 所有技能牌变为0费
- 打出时将其消耗
```

**能量收益**:
- 每张技能牌省下的费用 = 该牌原费用
- 假设卡组有5张1费技能牌 = 省5费

### 黑暗之拥效果

```
黑暗之拥（2费能力牌）:
- 每当你有一张牌被消耗，抽一张牌
```

**抽牌循环**:
- 消耗1张牌 = 抽1张牌
- 配合腐化实现：打出0费技能牌 → 消耗 → 抽牌 → 继续

### 能量效率分析

```
腐化+黑暗之拥启动成本: 2+2=4费

之后：
- 每张技能牌 = 0费 + 抽1牌
- 能量效率无限大
```

## Layer 2: 局部策略（流派特化）

### 启动顺序

```
理想启动:
1. 打出黑暗之拥
2. 打出腐化
3. 开始无限循环：打出0费技能牌 → 消耗 → 抽牌
```

### 关键问题

**问题**: 腐化沉底怎么办？

**解决方案**:
- 起手刷牌找腐化
- 如果没有腐化，先按普通流派打
- 腐化沉底时，可能需要重新评估策略

### 输出方式

| 输出牌 | 使用时机 |
|--------|----------|
| 全身撞击 | 高格挡时 |
| 巩固 | 翻倍格挡 |
| 普通攻击牌 | 腐化不影响攻击牌 |

## 关键牌详解

### 腐化 (Corruption)

```
费用: 2
效果: 所有技能牌变为0费，打出时将其消耗
升级: 费用变为1费
```

**使用要点**:
- 本流派核心
- 升级后1费更容易启动
- 注意：消耗后牌不进入弃牌堆

**价值**: 核心能力牌，价值★★★★★

### 黑暗之拥 (Dark Embrace)

```
费用: 2
效果: 每当你有一张牌被消耗，抽一张牌
升级: 费用变为1费
```

**使用要点**:
- 配合腐化实现抽牌循环
- 升级后更容易启动
- 也可以配合燃烧等其他消耗牌

**价值**: 抽牌引擎核心，价值★★★★★
