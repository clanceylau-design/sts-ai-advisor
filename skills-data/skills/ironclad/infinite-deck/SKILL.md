# 无限流 (Infinite Deck)

## 流派概述

**核心机制**: 日晷 + 双持 + 飞身踢 = 每回合无限循环，一回合击杀

**价值函数**: 无限输出 = 无视敌人血量

**风险等级**: 高（对卡组精简度要求极高）

**来源**: 攻略文档整理

**评价**: 高玩进阶玩法，最具挑战性和趣味性

---

## Layer 1: 数值计算（流派特化）

### 日晷机制

```
日晷（遗物）:
- 每当你洗牌时，获得2点能量
```

### 无限循环原理

```
核心循环:
1. 双持复制飞身踢（0费）
2. 飞身踢造成伤害并抽牌
3. 牌库抽空时洗牌
4. 触发日晷获得2点能量
5. 继续循环
```

### 能量计算

```
每次循环:
- 消耗: 0费（双持升级后）
- 获得: 2能量（日晷）
- 净收益: +2能量

理论上可以无限循环
```

---

## Layer 2: 局部策略（流派特化）

### 启动条件

```
必须条件:
1. 日晷遗物
2. 双持（升级后0费）
3. 飞身踢

卡组要求:
- ≤10张牌
- 最好≤5张牌
```

### 出牌顺序

```
无限循环:
1. 打出双持（0费）
2. 选择飞身踢复制
3. 打出飞身踢（造成伤害+抽牌）
4. 继续打出飞身踢
5. 牌库空→洗牌→日晷触发
6. 重复
```

### 注意事项

- 手牌上限10张，注意不要爆牌
- 飞身踢是2费，需要能量支持
- 升级双持变为0费是关键

## 关键牌详解

### 日晷 (Sundial)

```
类型: 遗物
效果: 每当你洗牌时，获得2点能量
```

**使用要点**:
- 无限流核心遗物
- 必须拿到才能玩无限流
- 提供无限能量来源

**价值**: 核心，价值★★★★★

### 双持 (Dual Wield)

```
费用: 1
效果: 将手牌中的一张攻击牌复制一张放入手牌
升级: 费用变为0费
```

**使用要点**:
- 升级后0费是关键
- 复制飞身踢
- 可以复制多次

**价值**: 核心，价值★★★★★

### 飞身踢 (Flying Knee)

```
费用: 2
效果: 造成8点伤害，抽1张牌
升级: 伤害提升至10点
```

**使用要点**:
- 伤害+抽牌一体
- 触发日晷的关键
- 无限输出来源

**价值**: 核心，价值★★★★★
