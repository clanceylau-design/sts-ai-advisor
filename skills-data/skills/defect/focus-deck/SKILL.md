# 集中流 (Focus Deck)

## 流派概述

**核心机制**: 通过多种方式获得高集中，所有充能球效果大幅提升

**价值函数**: 充能球效果 = 基础值 + 集中

**风险等级**: 中等

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 集中效果

```
每点集中提升:
- 闪电充能球: 伤害+1
- 冰霜充能球: 格挡+1
- 黑暗充能球: 积累伤害+1
- 等离子充能球: 能量+1
```

| 遗传算法 | +1/回合 | 永久增长 |

## Layer 2: 局部策略（流派特化）

### 集中获取顺序

```
优先级:
1. 碎片整理（永久+2）
2. 遗传算法（持续增长）
3. 偏差认知（短期+4，需要快速利用）
```

### 偏差认知使用

```
偏差认知特点:
- 获得4集中
- 每回合失去1集中
- 4回合后集中归零

使用策略:
- 配合快速充能球生成
- 偏差认知后立即打出充能球牌
- 4回合内最大化利用
```

### 遗传算法成长

```
遗传算法:
- 每回合获得1集中
- 配合机器学习（打出能力牌抽牌）

成长曲线:
回合1: +1集中
回合5: +5集中
回合10: +10集中
```
