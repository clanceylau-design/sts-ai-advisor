# 电球流 (Lightning Orb Deck)

## 流派概述

**核心机制**: 闪电充能球每回合对随机敌人造成伤害，无视防御

**价值函数**: 电球伤害 = 3 + 集中

**风险等级**: 中等

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 闪电充能球

```
闪电充能球:
- 每回合对随机敌人造成3点伤害
- 集中+1 = 伤害+1
- 最多同时拥有3个充能球（遗物可增加）
```

### 伤害计算

| 集中 | 单球伤害 | 3球总伤害 |
|------|----------|-----------|
| 0 | 3 | 9 |
| 2 | 5 | 15 |
| 4 | 7 | 21 |

### 双发效果

```
双发（1费）:
- 本回合你的充能球触发两次

效果: 伤害×2
```

## Layer 2: 局部策略（流派特化）

### 电球生成

| 卡牌 | 效果 |
|------|------|
| 球状闪电 | 7伤害+生成1电球 |
| 静电释放 | 受伤时生成1电球 |

### 雷霆打击

```
雷霆打击（3费）:
- 造成7点伤害
- 手牌中每有一个充能球，重复一次

3电球时: 7×4=28伤害
```

### 集中提升

| 卡牌 | 效果 |
|------|------|
| 碎片整理 | 获得2集中 |
| 偏差认知 | 获得4集中，每回合-1 |
