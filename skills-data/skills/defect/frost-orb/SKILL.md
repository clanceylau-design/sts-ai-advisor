# 冰球流 (Frost Orb Deck)

## 流派概述

**核心机制**: 冰霜充能球每回合提供格挡，集中提升格挡值

**价值函数**: 冰球格挡 = 2 + 集中

**风险等级**: 低

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 冰霜充能球

```
冰霜充能球:
- 每回合获得2点格挡
- 集中+1 = 格挡+1
- 最多同时拥有3个充能球（遗物可增加）
```

### 格挡计算

| 集中 | 单球格挡 | 3球总格挡 |
|------|----------|-----------|
| 0 | 2 | 6 |
| 2 | 4 | 12 |
| 4 | 6 | 18 |

### 双发效果

```
双发（1费）:
- 本回合你的充能球触发两次

效果: 格挡×2
```

| 卡牌 | 效果 |
|------|------|
| 碎片整理 | 获得2集中 |
| 偏差认知 | 获得4集中，每回合-1 |
| 遗传算法 | 每回合获得1集中 |

### 防御循环

```
理想循环:
1. 打出冰寒/冰川生成冰球
2. 冰球每回合自动提供格挡
3. 双发让格挡翻倍
4. 集中提升使格挡更高
```
