# 暗球流 (Dark Orb Deck)

## 流派概述

**核心机制**: 黑暗充能球每回合积累伤害，触发时造成爆发伤害

**价值函数**: 暗球伤害 = 6×回合数 + 集中×回合数

**风险等级**: 中等

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 黑暗充能球

```
黑暗充能球:
- 每回合积累6点伤害
- 集中+1 = 积累伤害+1
- 触发时造成所有积累的伤害
- 触发后重置
```

### 伤害积累

| 回合数 | 无集中伤害 | 集中+2伤害 |
|--------|-----------|-----------|
| 1 | 6 | 8 |
| 2 | 12 | 16 |
| 3 | 18 | 24 |
| 5 | 30 | 40 |

### 触发方式

```
多重释放（1费）:
- 触发最右侧充能球2次

双发（1费）:
- 本回合所有充能球触发两次

注意：暗球触发后伤害重置
```

## Layer 2: 局部策略（流派特化）

### 暗球生成

| 卡牌 | 效果 |
|------|------|
| 黑暗之爪 | 6伤害+生成1暗球 |
| 吸收 | 3格挡+生成1暗球 |

### 时机把握

```
积累策略:
1. 生成暗球后等待积累
2. 积累3-5回合获得高额伤害
3. 使用多重释放/双发触发

注意：
- 不要过早触发暗球
- 触发时机影响伤害量
- 多暗球可轮流积累
```

### 输出组合

```
爆发输出:
1. 暗球积累3回合=18伤害
2. 多重释放触发2次=36伤害
3. 双发让触发翻倍=72伤害
```
