# 刀贼流 (Knife Deck)

## 流派概述

**核心机制**: 无限刀刃 + 精准 = 持续获得高伤害小刀

**价值函数**: 小刀伤害 = 基础4 + 精准×3

**风险等级**: 中等

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 小刀机制

```
小刀（生成牌）:
- 费用: 0
- 效果: 造成4点伤害
- 使用后消耗
```

### 精准加成

```
精准（能力牌）:
- 小刀伤害+3
- 升级后+4
```

### 伤害计算

| 精准层数 | 小刀伤害 |
|----------|----------|
| 0 | 4 |
| 1 | 7 |
| 2 | 10 |
| 3 | 13 |

### 能量效率

```
无限刀刃（1费）:
- 每回合获得1张小刀
- 如果精准=2，小刀伤害=10
- 每回合等效输出=10伤害/1费
```

## Layer 2: 局部策略（流派特化）

### 启动顺序

```
1. 打出精准
2. 打出无限刀刃
3. 使用小刀输出
4. 终结技爆发
```

### 终结技使用

```
终结技（2费）:
- 造成6点伤害
- 本回合每打出一张攻击牌，伤害+2

配合小刀:
- 打出5张小刀后，终结技伤害=6+10=16
```

### 过牌策略

猎人过牌能力强：
- 后空翻: 4格挡+抽1牌
- 冲刺: 7格挡+抽1牌
- 杂技: 抽4张，弃1张

## 关键牌详解

### 无限刀刃 (Infinite Blades)

```
费用: 1
效果: 每回合开始时获得一张小刀
升级: 费用变为0费
```

**价值**: 刀贼核心，价值★★★★★

### 精准 (Accuracy)

```
费用: 1
效果: 小刀伤害+3
升级: 小刀伤害+4
```

**价值**: 伤害核心，价值★★★★★

### 终结技 (Finisher)

```
费用: 2
效果: 造成6点伤害，本回合每打出一张攻击牌，伤害+2
升级: 伤害提升
```

**价值**: 爆发核心，价值★★★★★
