# 毒贼流 (Poison Deck)

## 流派概述

**核心机制**: 叠毒 + 催化剂 = 毒层指数增长，无视敌人防御

**价值函数**: 毒伤害 = 毒层数（每回合触发）

**风险等级**: 中等

**来源**: 攻略文档整理

---

## Layer 1: 数值计算（流派特化）

### 毒机制

```
中毒状态:
- 敌人回合开始时，造成等同于毒层数的伤害
- 毒层数每回合-1
- 无视敌人格挡
```

### 毒来源

| 来源 | 毒层数 | 备注 |
|------|--------|------|
| 致命毒药 | 3 | 1费技能牌 |
| 弹跳药瓶 | 4 | 2费，+抽1牌 |
| 毒雾 | 2/回合 | 能力牌 |

### 催化剂翻倍

```
催化剂（1费）:
- 将敌人的中毒层数翻倍
- 升级后：翻三倍

爆发（0费）:
- 本回合下一张技能牌打出两次

组合:
爆发+催化剂 = 毒层×2×2 = 毒层×4
```

### 毒伤害计算示例

```
初始: 10层毒
催化剂: 20层
催化剂+爆发: 40层
催化剂+爆发+催化剂: 160层

每回合造成160伤害！
```

## Layer 2: 局部策略（流派特化）

### 叠毒顺序

```
理想顺序:
1. 致命毒药（3层）
2. 弹跳药瓶（+4层）
3. 催化剂（翻倍→14层）
4. 爆发+催化剂（翻倍→56层）
```

### 时机把握

- 毒需要回合数生效
- 不要急于爆发
- 先稳定叠毒，再催化剂翻倍

### 防御策略

毒贼需要存活：
- 使用后空翻、冲刺等格挡牌
- 猎人过牌能力强，找防御牌

## 关键牌详解

### 致命毒药 (Poison Stab)

```
费用: 1
效果: 给予3层中毒
升级: 给予4层中毒
```

**价值**: 叠毒核心，价值★★★★★

### 催化剂 (Catalyst)

```
费用: 1
效果: 将敌人的中毒层数翻倍
升级: 翻三倍
```

**价值**: 翻倍核心，价值★★★★★

### 爆发 (Burst)

```
费用: 0
效果: 本回合下一张技能牌打出两次
升级: 费用不变
```

**价值**: 配合催化剂，价值★★★★★
